http://github.com

és a hozzá tartozó kliens program kezelése.

A windowsos kliens programban csak azt állítsuk be, hogy mi a login a github szerveren
és hogy a gitshell karakteres felületet használjuk. Másra ez a kliens szerintem nem használható.

Karakteres felület:
===================

Új projekt kezdése
------------------
   1. github.com on létrehozunk egy "új_repostiory" -t (a felkinált initek közül nem kell egyik sem)
   2. $ git cd e/www/rootdir
   3. $ git init
   4. $ git add .
   5. $ git commit (a megjelenõ editorban KÖTELEZÕ LEIRÁST ADNI - nem # -al kezdödõ sorokban)
   6. $ git remote add "remotenev" http://github.com/utopszkij/"új-repostiory"
   7. $git -u "remotenév" master
   8. $ git branch "újbnév"
   9. $ git checkout "újbnév"
   10.$ git add .

Meglévõ projekt új "mérföldkõ"
------------------------------
   1. $ git cd e/www/rootdir
   2. $ git status  (ebbõl megtudjuk - többek között - a branch nevet)
   3. $ git commit (a megjelenõ editorban KÖTELEZÕ LEIRÁST ADNI - nem # -al kezdödõ sorokban)
   6. $ git remote add "remotenev" http://github.com/utopszkij/"repostiory"
   7. $ git -u "remotenév" "bnév"
